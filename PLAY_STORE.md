# Dollar Horde - Feature Summary for Store Listing

## Game Description (Max 4000 chars)
Survive endless hordes in this addictive auto-attacker! Fight, level up, and unlock powerful upgrades. Every wave brings tougher enemies — how long can you survive?

## Features
- ⚔️ **Auto-Attack Survival** - Move, shoot, survive. Simple controls, deep progression.
- ⬆️ **9 Unique Upgrades** - Speed Boost, Multi-Shot, Crit Chance, Armor, Regen, and more!
- 🧟 **Scaling Difficulty** - Face increasingly deadly hordes every 30 seconds.
- 💎 **Magnetic XP Pickup** - Collect orbs that fly toward you for satisfying progression.
- 🎵 **Procedural Audio** - Dynamic sound effects generated in real-time.
- 📱 **Mobile Optimized** - Touch joystick, portrait mode, one-handed play.
